# AgentTrap — Field Manual (website)

A static site that browses the AgentTrap v6 benchmark: every case file,
every model/framework run, every trajectory we captured.

## Layout

```
website_v2/
  index.html         landing / dossier
  cases.html         filterable catalog of 141 cases
  case.html          per-case detail (?id=N)
  runs.html          leaderboard of all runs
  run.html           per-run detail (?id=<sourceId>)
  traj.html          trajectory viewer (?run=...&case=N)
  assets/
    css/main.css
    js/{util,home,cases,case,runs,run,traj}.js
  data/              ← generated by scripts/build.py
  scripts/
    build.py
```

## Regenerate

```bash
python3 website_v2/scripts/build.py
```

Reads `cases/cases_v6.json`, `results/model_case_matrix_v6.json`, and the
matching `results/*/interactive_case_<N>.jsonl` trajectories. Writes
`website_v2/data/`. Redacts API keys and `/Users/...` paths.

## Preview locally

```bash
cd website_v2 && python3 -m http.server 8765
# open http://localhost:8765/
```

## Deploy on GitHub Pages

Target URL: `https://zhmzm.github.io/AgentTrap/`.

1. In the public release repo `zhmzm/AgentTrap`, create a `docs/` folder
   at repo root.
2. Copy `website_v2/{index.html,cases.html,case.html,runs.html,run.html,traj.html,assets,data}`
   into `docs/`.
3. Commit + push.
4. On GitHub: **Settings → Pages → Source: Deploy from a branch →
   Branch: `main` / folder: `/docs`**. Save.
5. The site comes up at `https://zhmzm.github.io/AgentTrap/` after one
   build cycle.
6. From your user site `zhmzm.github.io`, add a link card pointing to
   `https://zhmzm.github.io/AgentTrap/`.

Paths in the site are all relative, so the project subpath works without
configuration.

## Design

Field-manual / dossier aesthetic: Fraunces serif for display, JetBrains
Mono for body and data, parchment ground (`#EFE9DC`), ink (`#13110D`),
verdict colors keyed to outcome (oxblood `#B5341A` for attack succeeded,
forest `#3B4A2A` for blocked, amber `#9C7A1F` for utility incomplete, …).
No purple gradients, no glass cards, no Inter.
